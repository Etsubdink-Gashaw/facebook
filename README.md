# Facebook Clone Flutter App

A **Flutter-based Facebook clone** featuring:

- **Login page** with **Email/Password** and **Google authentication**  
- **Home page** for browsing posts and content  
- **Friends page** to view friend requests, suggestions, and your friends list  
- **Tab navigation** between Home, Friends, and other sections  

---

## Features

1. **Login / Authentication**
   - Email & password authentication via Firebase Auth  
   - Google Sign-In using Firebase Auth (mobile & web ready)  
   - Proper error handling for login failures  

2. **Home Page**
   - Placeholder for news feed or content  
   - Simple Flutter layout with AppBar and scrollable content  

3. **Friends Page**
   - Sectioned layout:
     - Friend Requests  
     - People You May Know  
     - All Friends  
   - Cards & ListTiles similar to Facebook UI  
   - Modular widgets for easy customization  

4. **Tab Navigation**
   - TabBar & TabBarView to switch between Home, Friends, and other pages  
   - Smooth swipeable pages  
   - Responsive on mobile and web
